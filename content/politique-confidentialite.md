# Qui sommes-nous ?
L’adresse de notre site est : [https://project-conquerors.com](https://project-conquerors.com).

# Données collectées

Les seules données collectées sont l'adresse IP ainsi que le user-agent de votre navigateur. Ces données sont collectées sur la base de notre intérêt légitime afin d'assurer le bon fonctionnement de ce site web ainsi que de faire des statistiques anonymisées. Ces données sont conservées pour une durée d'un mois avant d'être anonymisées.

# Contact

Vous pouvez exercer vos droits en adressant un mail à contact@project-conquerors.com en précisant dans le titre de votre mail qu'il s'agit d'une demande RGPD.
